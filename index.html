Equation


Number of Columns - First Column = Remaining Number of Columns

Remaining Number of Columns * Margin Spacing = Total Margin Spacing

Container Width - Total Margin Spacing = Remaining Width

Remaining Width / Total Number of Columns = Each Column Width

===========================================================================================

( $container_width - ( $margin_spacing * ( $number_of_columns - 1 ) ) ) / $number_of_columns

===========================================================================================

The Sass Mixin


@mixin equalSpaced( $number_of_columns, $margin_spacing , $container_width  ){

  width: ( $container_width - ( $margin_spacing  * ($number_of_columns - 1) ) ) / $number_of_columns;

  // Add Margin Left to Every Column
  margin-left: $margin_spacing ;

  // Remove Spacing from First Child
  &:first-child{
    margin-left: 0;
  }

}

// Useage

.col-2{
  @include equalSpaced(2, 10%, 100%);
}
